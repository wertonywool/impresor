# S.E.E.S. IMPRESOR - Persona 3 Edition

Solución inteligente de impresión optimizada para maximizar el uso del papel y evitar cortes automáticos en imágenes verticales.

## Características
- **Estética Persona 3:** Interfaz inspirada en el Dark Hour y el sistema S.E.E.S.
- **Protocolo Anti-Corte:** Las imágenes verticales se preservan sin recortes automáticos.
- **Optimización de Espacio:** CSS de impresión depurado para aprovechar cada milímetro.
- **Recorte Manual:** Integración con Cropper.js para ajustes precisos.
- **Soporte de Portapapeles:** Pega imágenes directamente (CTRL+V).

## Cómo usar
1. Abre `build_the_workers.html`.
2. Arrastra imágenes o pégalas.
3. Ajusta el número de columnas y el margen.
4. Presiona **🖨️ IMPRIMIR**.
